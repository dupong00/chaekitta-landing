Problem

열심히 읽었는데
왜 기억이 안 날까요?

에빙하우스 연구에 따르면 독서 후 1일이면 56%, 1주일이면 77%를 잊습니다. 문제는 독서 앱이 아무도 이걸 해결하지 않는다는 겁니다.

😓

기억이 안 남는다

열심히 읽어도 2주 후면 줄거리조차 흐릿해집니다. 읽은 것 같은데 아무것도 남지 않는 느낌.

📝

기록은 너무 귀찮다

독서 노트를 쓰려면 시간이 두 배로 걸립니다. 결국 흰 페이지만 남겨두고 포기하게 됩니다.

📱

기존 앱은 콘텐츠만

밀리의서재, 리디는 책을 파는 것에 집중합니다. 읽은 내용을 기억하게 도와주는 앱은 없습니다.

Solution

책잇다는 이렇게
다르게 작동합니다

1

책 등록

ISBN 스캔 또는 제목 검색으로 10초 안에 등록. 표지·저자·출판사가 자동으로 채워집니다.

2

AI가 먼저 질문합니다

챕터를 읽고 나면 AI가 "이 챕터에서 가장 인상 깊었던 부분은 무엇인가요?"처럼 먼저 말을 겁니다.

Claude Sonnet
3

대화하듯 짧게 답변

3~5문장이면 충분합니다. 노트 정리가 아니라 친구한테 말하듯이 하면 됩니다.

4

독서 노트 자동 생성

대화 내용을 AI가 자동으로 정리해 핵심 요약과 키워드로 구조화합니다. 직접 쓸 필요 없습니다.

Claude Haiku
5

망각 전에 복습 알림

1일 · 7일 · 30일 후, 기억이 흐려지기 직전에 해당 내용을 다시 보내드립니다. 과학적 타이밍입니다.
